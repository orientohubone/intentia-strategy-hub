Como salvar como PNG para Google:

  1. Faça print screen deste logotipo (120x120px)
  2. Cole em um editor de imagens (Paint, Photoshop, etc.)
  3. Recorte exatamente a área do logotipo
  4. Salve como PNG com nome "intentia-logo-120.png"
  5. Garanta que o arquivo tenha exatamente 120x120 pixels
  6. Tamanho máximo: 1MB (este arquivo será bem menor)

Formato aceito: JPG, PNG ou BMP
Resolução exigida: 120x120 pixels (quadrado)
